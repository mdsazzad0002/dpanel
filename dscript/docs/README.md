# dscript documentation

Documentation is intentionally consolidated into three files while retaining
the complete command and feature reference.

- [Install, commands, modules and configuration](installer.md)
- [Troubleshooting, recovery and repair](troubleshooting.md)

Quick discovery:

```bash
dpanel
dpanel help
dpanel list
dpanel module list
dpanel script list
dpanel doctor
```

`dpanel` opens the interactive menu. `dpanel help` prints the command reference.
Menu items include the matching command beside each option, and mutating actions
require typing `yes` before they run.
