First install

On a fresh server, /var/www is often owned by root. Use sudo git clone, then hand the cloned source back to your current user before running the installer.

sudo mkdir -p /var/www
cd /var/www
sudo git clone https://github.com/mdsazzad0002/dpanel.git dpanel-source
sudo chown -R "$USER:$USER" /var/www/dpanel-source
cd dpanel-source
sudo ./installer.sh chain install

If the file manager opens folders but cannot create, edit, upload, unzip, or delete files, run:

/var/www/dscript/scripts/fix-permissions.sh --all
